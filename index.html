Trainer help

Short reference for the hidden explanations.

Table playChoose which spots are You, Bot, or Empty. Only your spots affect the bankroll. Bot spots use basic strategy only and are mainly there to expose more realistic table cards.
CountingHi-Lo count: 2–6 = +1, 7–9 = 0, 10–A = −1. The dealer hole card is counted only after it is revealed. Hover the count boxes to peek at the hidden count.
AdviceBasic strategy is the by-the-book move for the current hand. Count-adjusted advice applies the built-in Hi-Lo deviation rules when the true count changes the decision.
Deviation trainerThe deviation table shows the index rules used by the app. For casino precision, indexes should be matched to the exact rules: deck count, S17/H17, surrender, double after split, penetration, and true-count rounding.
DrillsThe count drill uses a separate deck. Draw a card, update the running count in your head, enter it, and check. The strategy drill gives random dealer/player hands and checks the correct basic-strategy action.
